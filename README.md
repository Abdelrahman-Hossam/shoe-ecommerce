An eCommerce web app built with React.js that allows users to browse, filter, and view shoes dynamically.
This project was built to strengthen my React skills and showcase front-end development concepts like state management, component reusability, conditional rendering, and data fetching.

Features
- Browse a collection of shoes with detailed product info
- Filter and search products by category, size, or brand
- Dynamic UI rendering based on API data
- Smooth state management with useState and useEffect
- Add to cart (client-side functionality)
- Clean component-based structure
- Working on a responsive layout across all devices

How to run 
- npm run start
